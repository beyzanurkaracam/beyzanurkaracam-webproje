# beyzanurkaracam-webproje
web proje bilgi dokümanı
Proje Adı
Beyzanur Karaçam Web Proje

Projenin Amaçları

Bir web uygulaması oluşturarak, kendim hakkında bilgiler verip tanıtmak.
Ve bu web uygulamaları dersinde öğrendiklerimi sınayıp kendimi geliştirmek.


Projenin Özellikleri
Kullanıcılar web sitemden bilgi sahibi olabilir. 
Login ile siteye giriş yapabilir.
İletişim formundaki gerekli kontrol işlemlerini kontrol ettirebilir.

Kullanılan Teknolojiler
Front-end: HTML, CSS, JavaScript, React
Back-end:PHP
